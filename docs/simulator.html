Read it honestly: projected win totals compress toward .500 because the game model is deliberately conservative — MLB games really are near coin-flips (AUC ~0.55). So even a weak team keeps some playoff hope and the best teams don't run away. The ordering, though, tracks reality closely (r = 0.906). That gap between "confident spread" and "correct ranking" is the point.
