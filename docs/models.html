Methods

The Models

Thirteen models, each built in R and Python and gated on agreement, each anchored to a known baseball result. 13/13 pass the parity gate. Green means the two languages agree and the sanity check holds.

Model card index

M1 · Metric stabilityexactparity ✓
MethodCorrelation
Which hitting stats are skill vs. luck? We correlate each rate stat with itself year over year.
Result: K% repeats at r=0.82 and BB% at 0.73 (skill); BABIP only 0.30 (noise).
M2/M3 · xwOBA-over-expectedexactparity ✓
MethodLinear regression
Simple then multiple regression of batted-ball value on exit velocity and launch angle; residuals are hitter over/under-performance.
Result: Exit velo raises value (+0.008); angle is concave — there's an optimal window.
M4 · Catcher framingexactparity ✓
MethodLogistic GLM
Called-strike probability from pitch location; strikes stolen above expectation become framing runs.
Result: Best framer +32 runs, worst -21, over 3 seasons.
M5 · Strikeouts per startexactparity ✓
MethodPoisson regression
Strikeout counts per pitcher start — the shape behind K-total prop bets.
Result: League strikeout rate 0.218 per plate appearance.
M6 · Pitcher arsenalslabel-invariantparity ✓
MethodPCA + clustering
Pitch mix, velocity, spin, and movement compressed with PCA, then clustered into 'stuff' archetypes.
Result: Matched R↔Python by PC correlation + cluster ARI (1.00).
M7 · BABIP shrinkagelabel-invariantparity ✓
MethodMixed-effects
A random-intercept model shrinks each hitter's BABIP toward the mean — regression to the mean, quantified.
Result: ICC 0.27: only ~27% of BABIP is persistent skill.
M8 · Draft vs. productionexactparity ✓
MethodOLS (ethical scrape)
Draft pick (scraped politely from Wikipedia) vs. current MLB production for players who reached the majors.
Result: corr(pick, OPS) = -0.10, R² 0.009 — the draft is a crapshoot.
B1 · Pythagorean winsexactparity ✓
MethodOLS
Fit the exponent linking run ratio to win ratio, and the marginal runs-per-win.
Result: Exponent 1.73, ~11.0 runs per win.
B2 · Count effectsexactparity ✓
MethodRE24 aggregation
Run value and swing rate by ball-strike count, chaining RE24 down to the pitch.
Result: 3-0 worth +0.21 runs, 0-2 -0.10.
B3 · Aging curvesexactparity ✓
MethodQuadratic OLS
OPS as a quadratic function of age → peak-age estimate (cross-sectional; survivor bias noted).
Result: Concave curve peaking near age 30.
B4 · Markov inningsdistributionalparity ✓
MethodSimulation
Simulate half-innings from the base-out transition matrix; the simulated run expectancy must reconcile with RE24.
Result: Sim RE(empty,0) 0.50 reconciles RE24 0.51.
B5 · Streaky hittersdistributionalparity ✓
MethodPermutation test
Wald-Wolfowitz runs test on each hitter's game-by-game hit sequence, with a permutation null.
Result: Mean streakiness z = -0.12 — essentially indistinguishable from random.
GAME · Win probabilityexactparity ✓
MethodLogistic (sealed holdout)
Home win probability from pre-game rolling form, trained 2023–24 and evaluated once on the sealed 2025 season.
Result: Brier 0.247 beats the home-field baseline 0.249; AUC 0.55.
