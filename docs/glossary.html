Rate stats

On-base plus slugging — how often you reach base plus how much power you hit for.
How to read it: League average is ~.720; .900+ is All-Star, a full-season 1.000+ is an MVP line.

Statcast / expected

xwOBAHigher is better
Expected weighted on-base average — the value a batted ball 'should' produce given exit velocity and launch angle.
How to read it: Separates real contact quality from batted-ball luck; we rebuild the core relationship in M2/M3.
Exit velocity / launch angleHigher is better
How hard (mph) and at what vertical angle (degrees) a ball leaves the bat.
How to read it: Barrels live near ~98+ mph and 25–30°. Value is concave in angle — an optimal window (M2).
Pitch arsenal / stuffHigher is better
A pitcher's mix of pitch types and their velocity, spin, and movement.
How to read it: M6 clusters pitchers into archetypes (power, finesse, spin) from these features.

Run values

RE24 / run expectancyHigher is better
Expected runs from now to the end of the inning, for each of the 24 base-out states.
How to read it: The backbone of modern value stats. Bases empty, 0 outs ≈ 0.5 runs; loaded, 0 outs ≈ 2.3. Built from our own play-by-play.
Linear weights / run valueHigher is better
The average run value of an event (single, walk, HR, out…), derived from RE24.
How to read it: HR ≈ +1.4 runs, single ≈ +0.47, walk ≈ +0.31, strikeout ≈ −0.27 — the currency of wOBA, framing, and count value.
Framing runsHigher is better
Runs a catcher adds by getting borderline takes called strikes more than expected from location.
How to read it: Elite framers ≈ +10 to +20 runs a season (M4); real, measurable, invisible on TV.

Team & game

Win %Higher is better
Fraction of games a team wins.
How to read it: Home teams win ~54% of games — the baseline any prediction model must beat.
min 0.255median 0.506max 0.64
Leader Atlanta Braves 0.64Trailer Chicago White Sox 0.255
Pythagorean win %Higher is better
Expected win rate from runs scored and allowed: RS^k / (RS^k + RA^k).
How to read it: Run differential predicts a team's record better than its actual record does. We fit the exponent (B1) at ~1.73.
Luck (W − pythW)
Actual wins minus Pythagorean-expected wins — over/under-performance vs. run differential.
How to read it: Big luck rarely repeats — it's the gap between a team's record and what its runs earned.
Win probabilityHigher is better
The game model's estimated chance the home team wins, from leakage-safe pre-game form.
How to read it: On the sealed 2025 holdout it beats a home-field baseline on Brier — a small real edge (the game model).

Concepts

Metric stabilityHigher is better
How repeatable a stat is year to year. Skills stabilize fast; luck-driven stats don't.
How to read it: K% and BB% correlate ~0.8 year to year (skill); BABIP ~0.3 (noise) — M1.
Regression to the mean / ICCHigher is better
Extreme performances move back toward average; ICC is the share of a stat that is real skill vs. noise.
How to read it: BABIP's ICC is ~0.27 (M7) — mostly noise, so we shrink each hitter's BABIP toward the mean.
R↔Python parity gate
Every model is built twice — R and Python — and the build fails unless they agree.
How to read it: Proof the numbers aren't a one-tool artifact. Three tiers (exact / label-invariant / distributional).
